# Fina

Нейтральная версия личного финансового приложения, которой можно поделиться с друзьями.

## Что внутри

- счета;
- доходы и расходы;
- источники дохода;
- направления расходов;
- цели;
- анализ;
- встроенная инструкция в разделе `Анализ`;
- экспорт/импорт JSON;
- favicon и иконка для iPhone.

## Как пользоваться

1. Откройте вкладку `Операции`.
2. Нажмите `Добавить операцию`.
3. Выберите `Расход` или `Доход`.
4. Укажите сумму, название, дату, счёт, направление и категорию.
5. Для дохода выберите источник: основной доход, доп. доход, проекты, онлайн-доход или инвестиции.
6. На вкладке `Главная` смотрите баланс, доходы, расходы, свободные деньги и дневной лимит.
7. На вкладке `Анализ` смотрите структуру доходов/расходов и делайте бэкап.

## Как обновить GitHub Pages

Загрузите в корень репозитория:

```text
index.html
manifest.webmanifest
favicon.png
apple-touch-icon.png
icon-192.png
icon-512.png
README.md
```

## Важно

Данные хранятся локально в браузере устройства.

Перед тем как делиться ссылкой, не вносите туда личные финансовые данные.

Для чистого запуска используется ключ хранения:

```text
fina_state_v3
```
